# Frontend SPA

This folder contains the React + TypeScript + Vite SPA served by the Nginx runtime image.

Target responsibilities:

- client-side routing with React Router;
- generated TypeScript API client from OpenAPI;
- remote state with TanStack Query;
- forms with React Hook Form and Zod;
- unit/component tests with Vitest and React Testing Library;
- representative browser tests with Playwright.
